int32_t Light::getState ( int32_t *  state)

Read state from server.

Parameters
state- the pointer to store the state which is one of LIGHT_STATE_OFF and LIGHT_STATE_ON.
Return values
0- success.
ERR_INVALID_PARAMETER- state is invalid.
ERR_NO_RESPONSE- no response package from server.
ERR_NO_EXPECT- no state information expected in response package from server.
Usage: a snippet following
ret = obj.getState(&state);
if (state == LIGHT_STATE_ON)
{
digitalWrite(light_pin, HIGH);
}
else if (state == LIGHT_STATE_OFF)
{
digitalWrite(light_pin, LOW);
}
else
{
// do something else
}
